How to contribute to MPC wiki?

The wiki is hosted on github pages so to contribute to the wiki you will have to commit on github (see below). Be sure to check our discord or you can use BETA our editor

Requirements

  • html knowledge
  • MPC knowledge
  • Github account
  • ability to use git (I would not cover that here)
  • (Optional but preferred) being in our discord

Step 1: fork the main directory

find a fork button somewhere

Step 2: clone the forked directory

input git clone and the path to your wiki's fork

Step 3: make your changes

edit create and share your knowledge

Step 4: commit your changes to the main directory

create a pull request
